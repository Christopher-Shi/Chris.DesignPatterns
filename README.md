# Chris.DesignPatterns
### 面向对象设计的三大原则

- 哪里有变化，我就封装哪里
- 面向抽象编程，细节和高层实现都要依赖抽象
- 多组合，少继承 

### 根据这三大原则又衍生出来6个具体的原则

- 单一职责原则
- 里氏替换原则
- 依赖倒置原则
- 接口隔离原则
- 迪米特法则
- 开闭原则
